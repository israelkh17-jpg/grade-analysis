📥 NEIS에서 정오표 다운로드 방법

💡 안내: NEIS(나이스) 교무업무 시스템에서 "지필평가 교과목별 학생답 정오표"를 반별로 다운로드한 후, 본 도구에 업로드하세요.

1

지필평가 조회 메뉴 진입

NEIS 좌측 메뉴에서 교과담임 탭을 펼친 후 지필평가 조회/통계 → 지필평가 조회를 클릭합니다.

📌 좌측 메뉴 트리에서 "지필평가 조회" 진입
2

"교과목별 학생답 정오표" 선택 및 조회

상단 탭에서 "교과목별학생답정오표"를 선택한 후 다음 정보를 입력합니다:

  • 학년도: 해당 연도
  • 학기: 1학기 / 2학기
  • 고사: 1학기 중간고사, 기말고사 등
  • 학년: 학년 선택
  • 과목: 분석할 과목 선택

모두 입력한 후 [조회] 버튼을 클릭하면 학생별 정오표가 화면에 나타납니다.

3

저장 → "XLS data" 형식으로 다운로드

화면 상단의 [저장] 버튼을 누르면 형식 선택 메뉴가 나타납니다.

여러 형식 중 📊 XLS data를 선택하여 엑셀 파일로 다운로드하세요.

📌 저장 형식 선택 화면 (XLS data 선택)
⚠️ 중요: 모든 반(1반~9반)을 각각 따로 다운로드해야 합니다. 반을 변경하면서 [조회] → [저장] 과정을 반복하세요.
4

본 도구에 업로드

다운로드한 모든 반의 정오표 엑셀 파일을 한 번에 본 도구에 드래그&드롭하면 자동으로 분석이 시작됩니다.

파일명에 반 번호(101, 102, ...)가 포함되어 있으면 자동으로 반이 인식됩니다.

📄 원본 PDF 가이드를 직접 보시려면 여기를 클릭하세요.

📋 문항정보표 등록

문항정보표 데이터를 입력하면 내용 영역별 분석, 성취기준별 도달도, 출제 의도 vs 실제 난이도 검증이 가능해집니다.

학교에서 출력한 "문항정보표" PDF 파일을 그대로 업로드하세요. 자동으로 표를 인식하여 분석합니다.

📌 일반적인 학교 양식의 문항정보표 (선택형/서답형 표 포함)를 지원합니다.
📌 PDF에서 문항번호, 내용영역, 성취기준 코드, 난이도, 배점, 정답을 자동 추출합니다.

엑셀에서 문항정보표를 복사한 후 아래에 붙여넣으세요. (탭 또는 쉼표로 구분된 데이터 모두 지원)

형식: 문항번호 / 내용영역 / 성취기준 / 난이도 / 배점 / 정답
예시: 1 과학의 기초 [10통과1-01-02] 쉬움 3.1 2

문항정보표 엑셀 파일(.xlsx) 또는 CSV 파일을 업로드하세요.

파일의 첫 행은 헤더(컬럼 이름)여야 합니다.

🤖 AI 성취수준 분석

학생을 선택하면, 출제원안지·문항정보·응답 결과를 바탕으로 AI가 내용 영역별 성취수준 진술문을 생성합니다.

💡 사용안내

1

나이스에서 문항정보표(PDF) 다운로드

NEIS에서 문항정보표 PDF를 다운받아 사용하세요.

📌 준비물: 문항정보표 + 출제원안지

2

샘플 데이터로 분석 예시 보기

샘플 데이터를 선택하면 분석 예시를 확인하실 수 있습니다.

📌 1학기 중간 ○○고의 자료입니다. 가명 처리되었으니 안심하셔도 됩니다.

3

본인 자료를 직접 업로드

대시보드 상단의 "📄 출제원안지 등록", "📋 문항정보 등록"에서 직접 자료를 업로드하시면 분석 결과를 더 정밀하게 확인하실 수 있습니다.

📌 샘플 데이터를 그대로 사용하셔도 괜찮습니다.

!

AI 성취수준 분석은 조금만 사용해 주세요

AI 분석은 호출 시마다 비용이 발생합니다. 꼭 필요한 학생 위주로 사용해 주세요. 🙏

📌 문항정보표와 출제원안지를 모두 올려주셔야 AI 분석 버튼이 활성화됩니다.

📌 첫 학생 분석 후 5분 이내 같은 시험으로 다른 학생을 분석하면 캐시로 더 저렴해집니다.

📄 출제원안지 등록

출제원안지(시험지) PDF를 등록하면, 문항정보표·학생 성적과 함께 AI가 내용 영역별 성취수준 진술문을 자동으로 생성합니다.

📌 분석 작동 조건
1. 출제원안지 PDF (이 모달에서 등록)
2. 문항정보표 (📋 버튼에서 등록)
3. 학생 성적 데이터 (1반~9반 정오표)
세 가지가 모두 등록되면 AI 분석 버튼이 활성화됩니다.

시험지 원본 PDF를 업로드하세요. Gemini 멀티모달 AI가 문항 내용을 직접 읽어 분석합니다.

📌 지원 형식: PDF (최대 15MB)
📌 업로드된 파일은 브라우저에만 저장되며, AI 분석 시에만 API 서버로 전송됩니다.
